# javaEstudos

Repositório estruturado para o armazenamento e organização dos códigos desenvolvidos durante as aulas teóricas e resoluções de exercícios práticos do DevDojo.

---

## Estrutura e Escopo

O repositório armazena a evolução prática em lógica de programação e fundamentos da linguagem Java. O código é segmentado logicamente para garantir legibilidade e separação de conceitos:

* **Teoria:** Implementações baseadas no acompanhamento do conteúdo programático.
* **Exercícios:** Resoluções independentes de desafios aplicados.

---

## Ambiente de Desenvolvimento

* **Linguagem:** Java
* **Runtime:** Eclipse Temurin JDK 25 (LTS)
* **IDE:** IntelliJ IDEA
* **Controle de Versão:** Git / GitHub

---

## Estrutura de Diretórios

* `src/` : Diretório raiz do código-fonte.
    * `devdojo/` : Pacotes destinados aos exemplos e sintaxe das aulas.
    * `exercicios/` : Resoluções de problemas e algoritmos propostos.

---
